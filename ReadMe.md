# Sistema Académico em PHP

## Descrição
Aplicação web desenvolvida em PHP com MySQL para gestão académica.

## Funcionalidades
- Login e logout
- Registo de alunos com fotografia
- Gestão de cursos e unidades curriculares
- Pedidos de matrícula/inscrição
- Pedidos de mudança de curso
- Pedidos de certificado de habilitações
- Criação de pautas
- Lançamento e edição de notas

## Perfis
- Aluno
- Funcionário
- Gestor

## Tecnologias
- PHP
- MySQL
- PDO
- HTML/CSS
- Bootstrap

## Como executar
1. Colocar o projeto na pasta htdocs do XAMPP
2. Importar o ficheiro sql/database.sql no phpMyAdmin
3. Confirmar os dados da ligação no ficheiro config/database.php
4. Abrir no browser:
   http://localhost/academico_simples/
